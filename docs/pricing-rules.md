# Árazási szabályok

## Jelenlegi modell
Az árazás adatvezérelt és később admin felületből szerkeszthető.

## Követelmények
- A foglalási ár részletes pillanatképe a foglalás mentésekor rögzítve lesz.
- Minden árképzési logika szerveroldalon történik.
- A pricing modul legyen különválasztva a booking modulhoz.
