# iCal sync

## Cél
Az iCal import/export külön modulban kerül kezelése, hogy a kompatibilitási logika ne keveredjen a foglalási szabályokkal.

## Tervezett feladatok
- iCal export a foglalási adatokból.
- iCal import a külső naptárból.
- Személyes adatok kizárása az exportból.
- A modul egyértelmű interface-al szolgáljon a domain és az integráció között.
